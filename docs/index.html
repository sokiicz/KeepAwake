What it does

Everything you need, nothing you don't

Simple to run, powerful under the hood.

🔒

Prevents Sleep & Lock

Keeps your session alive using silent F15 keypresses by default — invisible to applications but effective at resetting the idle timer.

🧠

Smart AFK Simulation

Periodically pauses activity for a random duration to mimic natural usage patterns. Prevents you appearing "always online" 24/7.

🕐

Scheduled Hours

Set a start and end time so prevention only runs during your working hours — automatically stops outside those hours.

🖥

System Tray

Runs silently in the background. Right-click the tray icon to show the window or quit. No console window, no clutter.

⌨

Global Hotkeys

Toggle prevention or show/hide the window from anywhere with Ctrl+Alt+K and Ctrl+Alt+H — no need to find the window.

🔥

Multiple Methods

Choose your prevention technique: F15 key (silent default), F16 key, Scroll Lock, or a 1-pixel mouse nudge.

📊

Session Statistics

Tracks total prevention actions and uptime per session, stored locally in a lightweight SQLite database.

🌓

Dark & Light Theme

Switch between a sleek dark mode and a clean light mode from within the app — no restart needed.

Get started

Installation

Up and running in under a minute.

1

Clone or download

Clone the repository, or download the ZIP and extract it anywhere.

git clone https://github.com/sokiicz/KeepAwake.git
cd KeepAwake
2

Install dependencies

Requires Python 3.8+. Install all packages with one command:

pip install customtkinter pystray pillow pynput psutil
3

Launch

Easiest way: double-click START_Fixed_KeepAwake_Pro.bat — it installs dependencies and starts the app automatically.

Or run directly:

pythonw keepawake_pro.py

Use pythonw (not python) to suppress the console window.

Smart AFK Simulation

Look human, not like a script.

After 5 minutes of continuous prevention, KeepAwake may randomly pause for 30–120 seconds, simulating you stepping away from your keyboard. This prevents you appearing "always active" in monitoring tools, games, or chat applications.
